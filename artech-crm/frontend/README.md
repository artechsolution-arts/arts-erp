# Artech CRM Frontend

Vue.js frontend for Artech CRM. Communicates with the Artech backend via REST API.

## Tech Stack

- Vue 3 (Composition API)
- Vite
- frappe-ui component library
- Tailwind CSS

## Setup

```bash
yarn install
yarn dev
```

Requires the Artech bench running at `http://localhost:8000`. See [parent README](../README.md) for full setup.

## Build

```bash
yarn build
```
